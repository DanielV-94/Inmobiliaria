# Variante A — Black Label (Single Page)

Propuesta premium con enfoque cinematográfico y scrolltelling.

## Incluye

- Loader de entrada premium
- Animaciones con GSAP + ScrollTrigger
- Smooth scroll con Lenis
- Fondo ambiental Three.js en hero
- SVG animado (line draw) + sección scrolltelling pinneada
- Cards con tilt/interacción de lujo

## Abrir

Servir la carpeta en un servidor estático y abrir `index.html`.
